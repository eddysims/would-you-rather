# Modal

A `Modal` is an overlay that allows users to view, information that doesn't require a page to be built. It also prevent users from interacting with the rest of the page until a specific action is taken.

## Usage

```tsx
import { Modal } from "@/components/Modal";
```

<ArgsTable component={Modal} />

## Example

<Canvas>
  <Story name="Default">
    {(args) => {
      const [open, setOpen] = useState(true);
      return (
        <>
          <button onClick={toggleModal}>Toggle Modal</button>
          <Modal open={open} onClose={toggleModal}>
            Modal content
          </Modal>
        </>
      );
      function toggleModal() {
        setOpen(!open);
      }
    }}
  </Story>
</Canvas>
