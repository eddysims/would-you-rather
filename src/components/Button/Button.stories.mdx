# Button

A `Button` is a common element that allows a user to perform an action.

## Usage

```tsx
import { Button } from "@/components/Button";
```

## Props

<ArgsTable component={Button} />

## Example

<Canvas>
  <Story
    name="Default"
    args={{
      title: "Shop me some coffee",
      onClick: () => alert("mmm... coffee"),
    }}
  >
    {(args) => <Button {...args} />}
  </Story>
</Canvas>

## Loading state

You can indicate that a button is a loading state by setting the `loading` prop.

When a button is `loading` it will not be interactive to the user.

<Canvas>
  <Story
    name="Loading state"
    args={{
      title: "Load the coffee machine",
      loading: true,
      onClick: () => alert("coffee loading"),
    }}
  >
    {(args) => <Button {...args} />}
  </Story>
</Canvas>

## Submitting a form

A `Button` can be used to submit a `Form` by using the `isSubmit` prop.

<Canvas>
  <Story
    name="Submitting a form"
    args={{
      title: "Order more coffee",
      isSubmit: true,
      onClick: () => alert("Coffee has been ordered"),
    }}
  >
    {(args) => <Button {...args} />}
  </Story>
</Canvas>
