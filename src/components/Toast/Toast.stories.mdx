# Toast

`Toast` is used to show a short term notification.

The `Toast` component exposes a `ref` with a `notify` method attached. To use the `Toast` you will want to call the `notify` function.

```tsx
import { useRef } from "react";
import { Toast } from "@/components/Toast";

export function Component() {
  const toastRef = useRef();
  return (
    <>
      <button onClick={() => toastRef.current.notify({...})}>Notify</button>
      <Toast ref={toastRef} />
    </>
  )
}
```
