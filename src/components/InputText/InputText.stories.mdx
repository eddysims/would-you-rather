# InputText

## Usage

```tsx
import { InputText } from "@/components/InputText";
```

## With validations

The `InputText` component uses [React Hook Form](https://react-hook-form.com/) to validate fields.

<Canvas>
  <Story
    name="With validations"
    args={{
      label: "Enter Your Name",
      validations: {
        minLength: {
          value: 3,
          message: "Thats an awfully short name",
        },
        required: "You need a name",
      },
    }}
  >
    {(args) => <InputText {...args} />}
  </Story>
</Canvas>

## With an initial value

You can set the initial value of an input useing the `initialValue` prop.

<Canvas>
  <Story
    name="With an initial value"
    args={{
      label: "Who the heck are ya?",
      initialValue: "Eddy Sims",
    }}
  >
    {(args) => <InputText {...args} />}
  </Story>
</Canvas>
